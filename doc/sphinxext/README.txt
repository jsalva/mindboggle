===================
 Sphinx Extensions
===================

These are a few sphinx extensions we are using to build the Mindboggle
documentation::

* From matplotlib:
  * inheritance_diagram.py
  * ipython_console_highlighting.py
  * mathmpl.py
  * only_directives.py
  * plot_directive.py

* From numpy:
  * docscrape.py
  * docscrape_sphinx.py
  * numpydoc.py
